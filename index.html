Exploring Web Development

September 24, 2025 by Daniel Bingham

Lately, I’ve been diving headfirst into web development, and it’s been a fascinating shift from my usual deep dives into algorithms and system-level programming. There’s something incredibly satisfying about seeing code come to life in a browser and not just as logic, but as layout, color, and interaction. I started with HTML and CSS, building static pages and experimenting with different styles, but once I added JavaScript to the mix, everything changed. Suddenly, I could make things move, respond, and adapt. It felt like unlocking a new layer of creativity, one that blends design with logic in a way that’s both intuitive and challenging.


What’s surprised me most is how much thought goes into user experience. As a computer science student, I’m used to thinking about efficiency and correctness, but web dev forces me to consider clarity, accessibility, and emotion. I’ve been working on a personal blog to document my journey and not just the technical stuff like flexbox layouts or API calls, but also the mindset shifts. Each post is a reflection of what I’ve learned and where I’ve stumbled. It’s become more than a portfolio; it’s a space where I can connect my love for clean code with storytelling and design.
